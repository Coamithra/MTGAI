# Image Generation Test Prompts

10 standardized prompts for evaluating image generation models (SDXL, Flux, DALL-E, Midjourney) against MTG card art requirements.

**Art box aspect ratio:** ~1.56:1 (width > height). Generate at 1536x1024 or similar wide format.
**All prompts include anti-text directives** to suppress text/watermark artifacts.

## Generation Parameters

When running the test suite, use these settings where supported:

| Parameter | Value | Notes |
|-----------|-------|-------|
| **Resolution** | 1536 x 1024 (3:2) | Closest standard size to 1.56:1 art box ratio |
| **CFG Scale** | 7.0 (SDXL), 3.5 (Flux) | Defaults; adjust if needed |
| **Steps** | 30 (SDXL), 20 (Flux schnell), 30 (Flux dev) | Standard quality steps |
| **Sampler** | DPM++ 2M Karras (SDXL), Euler (Flux) | Recommended defaults |
| **Seed** | Use same seed across models where possible | Enables fairer comparison |
| **Negative prompt** | As specified in optimized versions | SDXL only; Flux does not use negative prompts natively |

## Scoring Template

For each model x prompt combination, score 1-5 on each criterion.

**Scoring scale:**
- **5** = Excellent -- professional MTG art quality, no issues
- **4** = Good -- minor imperfections, would still work on a card
- **3** = Acceptable -- noticeable issues but usable with minor edits
- **2** = Below average -- significant issues, would need major rework or regeneration
- **1** = Unusable -- wrong subject, severe artifacts, completely off-prompt

### Scoring criteria (from Section 2.2 of Phase 0B plan)

| Criterion | Weight | What to evaluate |
|-----------|--------|------------------|
| Fantasy Quality | 30% | Does it look like professional fantasy card game art? Composition, detail, lighting |
| Style Consistency | 25% | Could this sit alongside the other 9 images in a coherent set? |
| Prompt Adherence | 15% | Did the model render what was actually described? |
| Aspect Ratio | 10% | Clean output at 1536x1024 without distortion or awkward cropping? |
| Resolution | 10% | Sharp details at native resolution? Holds up at 300 DPI print size? |
| Artifacts | 5% | Hands, faces, anatomy, floating objects, weird blending? |
| Speed | 5% | Generation time (record in notes) |

### Weighted Score Calculation

```
Weighted Score = (Fantasy * 0.30) + (Consistency * 0.25) + (Adherence * 0.15) +
                 (AspectRatio * 0.10) + (Resolution * 0.10) + (Artifacts * 0.05) +
                 (Speed_normalized * 0.05)
```

For Speed normalization: fastest model gets 5, slowest gets 1, others interpolated linearly.
