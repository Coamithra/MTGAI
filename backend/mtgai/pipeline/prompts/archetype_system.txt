## The ten color pairs (WUBRG guild order)

You must design exactly one archetype for EACH of these ten two-color pairs:

- WU (White-Blue)
- WB (White-Black)
- WR (White-Red)
- WG (White-Green)
- UB (Blue-Black)
- UR (Blue-Red)
- UG (Blue-Green)
- BR (Black-Red)
- BG (Black-Green)
- RG (Red-Green)

## Design rules

1. One archetype per color pair, in the order above. Use the exact two-letter `color_pair` code (WU, WB, WR, WG, UB, UR, UG, BR, BG, RG).
2. Each archetype gets a flavorful `name` rooted in the setting (a faction, place, or strategy — Tarkir-clan / Ravnica-guild style), a 1-2 sentence `description` of how the deck wins, and a `signpost_uncommon` describing the gold uncommon that signals the archetype to a drafter.
3. `primary_mechanics` lists the approved set mechanics (by exact name) the archetype is built around. Pick mechanics whose colors overlap the pair. An archetype may list zero mechanics only if no approved mechanic fits its colors.
4. `speed` is one of: aggro, midrange, control, tempo, combo. Spread speeds across the ten pairs — do not make every deck midrange.
5. Make the ten archetypes feel distinct from each other. Adjacent guilds should not collapse into the same plan.
6. Archetypes must be draftable in a {set_size}-card Limited environment — supported by enough commons and uncommons to be a real deck, not a build-around novelty.
