# Font Assets

Open-source alternatives to MTG's proprietary fonts, for card rendering.

## Font Selection

| Purpose | MTG Original | Our Font | License | Weight Range |
|---------|-------------|----------|---------|--------------|
| Card name, type line | Beleren | **Cinzel** | OFL | 400-900 |
| Rules text, flavor text | MPlantin | **EB Garamond** | OFL | 400-800 (+ italic) |
| P/T, collector info | Gotham | **Montserrat** | OFL | 100-900 (+ italic) |

## File Structure

All fonts are **variable TTF** files (single file contains all weights):

- `cinzel/Cinzel-Variable.ttf` — wght axis 400-900
- `eb-garamond/EBGaramond-Variable.ttf` — wght axis 400-800
- `eb-garamond/EBGaramond-Italic-Variable.ttf` — wght axis 400-800
- `montserrat/Montserrat-Variable.ttf` — wght axis 100-900
- `montserrat/Montserrat-Italic-Variable.ttf` — wght axis 100-900
